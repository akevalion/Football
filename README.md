# Futbol

Roassal3 soccer/futbol/football demo

## Installation
 
Use the following incantation on you pharo image

```st
Metacello new
	baseline: 'Futbol';
	repository: 'github://akevalion/Football';
	onConflictUseIncoming;
	load.
```
